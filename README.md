# Advise.me

An academic assistant platform for college students. Ask questions about degree requirements, course offerings, and academic policies. Get instant, accurate answers instead of waiting days for a response. 

## Stack
 
- **Frontend** — Next.js
- **Backend** — FastAPI (Python)
- **Database** — PostgreSQL with pgvector
- **Auth** — NextAuth scoped to `@pitt.edu` emails
